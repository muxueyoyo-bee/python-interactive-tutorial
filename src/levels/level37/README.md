# 第三十七关: numpy 数组运算

NumPy 的核心优势是向量化运算 -- 不用写循环就能对整个数组执行数学运算.

## 向量化运算

```python
import numpy as np
a = np.array([1, 2, 3, 4, 5])
print(a * 2)        # [2 4 6 8 10]  每个元素乘2
print(a + 10)       # [11 12 13 14 15]  每个元素加10
print(np.mean(a))   # 3.0   求平均值
```

## 常用函数

| 函数 | 含义 |
|------|------|
| np.mean() | 平均值 |
| np.sum() | 求和 |
| np.max() | 最大值 |
| np.min() | 最小值 |
| np.std() | 标准差 |

向量化运算比 Python 循环快 10-100 倍.
