# 第二关：变量与赋值

在这一关，你将学习如何用**变量**来存储数据。

## 什么是变量？

变量就像一个**带标签的盒子**，你可以把数据放进去，并通过标签来访问它。

```
变量名 → 数据
name  → "小明"
age   → 18
```

## 赋值语句

使用 `=` 号将值赋给变量：

```python
name = "小明"    # 字符串变量
age = 18         # 整数变量
score = 95.5     # 浮点数变量
is_pass = True   # 布尔变量
```

> `=` 不是数学中的"等于"，而是**赋值**操作——把右边的值放入左边的变量中。

## print() 输出多个值

`print()` 可以同时输出多个值，用**逗号**分隔：

```python
name = "小明"
age = 18
print(name, age)
# 输出: 小明 18
```
