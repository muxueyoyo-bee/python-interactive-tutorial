# 第二十四关: 正则表达式

正则表达式(regex)是一种模式匹配语言, 用于在文本中查找/提取特定格式的内容.

## re 模块

```python
import re

text = "联系邮箱: abc@example.com, 电话: 13800138000"

email = re.search(r"[\w.]+@[\w.]+", text)
phone = re.search(r"1\d{10}", text)
```

## 常用模式

| 模式 | 含义 |
|------|------|
| \\d | 数字 |
| \\w | 字母/数字/下划线 |
| + | 一个或多个 |
| {n} | 恰好 n 个 |
