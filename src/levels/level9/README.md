# 第九关: 字典的增删改查

> 前置回顾: 第8关 字典创建和访问, 第6关 列表的增删改查

## 增/改 -- 同样的语法

```python
d = {"a": 1, "b": 2}
d["c"] = 3       # 键 "c" 不存在 -> 新增
d["a"] = 10      # 键 "a" 已存在 -> 修改
print(d)          # {'a': 10, 'b': 2, 'c': 3}
```

> 和第6关列表对比: 列表用 append/insert, 字典直接用 [] 赋值即可。

## 删除 -- 三种方式

```python
d = {"a": 1, "b": 2, "c": 3}

del d["b"]        # 直接删除
v = d.pop("c")    # 删除并返回被删的值, v=3
```

## 安全访问

```python
d = {"name": "小明"}
print(d.get("gender"))        # None (安全)
print(d.get("gender", "男"))   # "男" (设默认值)

# 先检查再访问
if "gender" in d:
    print(d["gender"])
```

## 常见错误

- 用不存在的键: KeyError, 用 get() 避免
- 字典是可变对象: `d2 = d1` 后改 d2 会影响 d1

## 你的任务

从 {"a":1,"b":2} 开始: 新增 "c"=3 -> 修改 "a"=10 -> 删除 "b" -> 打印最终字典。

预期输出:
```
{'a': 10, 'c': 3}
```
