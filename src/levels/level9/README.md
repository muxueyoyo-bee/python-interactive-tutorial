## # 第九关: 字典的增删改查
字典是可变的, 可以动态添加, 修改和删除键值对.

## 增加和修改

```python
d = {"a": 1, "b": 2}
d["c"] = 3      # 新增
d["a"] = 10     # 修改
```

## 删除

```python
d = {"a": 1, "b": 2, "c": 3}
del d["b"]       # 删除键 "b"
d.pop("c")       # 弹出并返回
```

## 检查键是否存在

```python
print("a" in d)       # True
print("z" in d)       # False
```

## 你的任务

从 {"a": 1, "b": 2} 开始, 添加键 "c"=3, 修改 "a"=10, 删除 "b", 打印最终字典.

预期输出:
```
{'a': 10, 'c': 3}
```
