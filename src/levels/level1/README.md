## Python 是什么？

Python 是一门**简洁、优雅、功能强大**的编程语言。它被广泛应用于：
- 数据分析与人工智能
- Web 开发
- 自动化脚本
- 科学计算

## print() 函数

`print()` 是 Python 最基础的函数，用于**在屏幕上输出内容**。

### 基本语法

```python
print("你要输出的内容")
```

### 示例

```python
print("Hello, World!")
# 输出: Hello, World!

print("你好，Python！")
# 输出: 你好，Python！
```

> **注意**：文本需要用英文双引号 `" "` 或单引号 `' '` 包裹起来。
