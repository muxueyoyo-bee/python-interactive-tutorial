## # 第十一关: if 条件判断
if 语句是程序分支的基础 -- 根据条件执行不同的代码.

## 基本语法

```python
if 条件:
    执行代码
elif 另一个条件:
    执行另一段代码
else:
    以上都不满足时执行
```

> 注意: `:` 冒号不能忘! 缩进(4个空格)表示代码块.

## 比较运算符

| 符号 | 含义 |
|------|------|
| > | 大于 |
| < | 小于 |
| >= | 大于等于 |
| <= | 小于等于 |
| == | 等于 |
| != | 不等于 |

## 示例: 分数评级

```python
score = 85

if score >= 90:
    print("优秀")
elif score >= 60:
    print("及格")    # 85分走这里
else:
    print("不及格")
```
