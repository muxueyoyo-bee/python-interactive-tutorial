# 第十一关: if 条件判断

> 前置回顾: 前10关代码都是"顺序执行" -- 从上到下一行不漏。现在学"分支"。

## 三种执行方式

1. **顺序**: 一行一行执行 (之前所有代码)
2. **分支**: 根据条件选择性执行 (本关)
3. **循环**: 重复执行某段代码 (第12/13关)

## if 基本语法

```python
if 条件:
    条件成立时执行
elif 另一个条件:
    上一个不成立但本条件成立时执行
else:
    以上都不成立时执行
```

关键: `:` 冒号不能忘, 缩进(4空格)表示代码块。

## 比较运算符

| 符号 | 含义 | 示例 |
|------|------|------|
| `>` | 大于 | `5 > 3` -> True |
| `<` | 小于 | `5 < 3` -> False |
| `>=` | 大于等于 | `5 >= 5` -> True |
| `==` | 等于 | `5 == 5` -> True |
| `!=` | 不等于 | `5 != 3` -> True |

> 重点: `==` 是比较, `=` 是赋值。初学者最容易混淆!

## 示例: 分数评等级

```python
score = 85

if score >= 90:
    print("优秀")
elif score >= 60:
    print("及格")    # 85分走这里
else:
    print("不及格")
```

条件顺序很重要 -- 如果把 `>=60` 放最前面, 85分只会输出"及格"。

## 逻辑运算符

```python
age = 25
has_ticket = True

if age >= 18 and has_ticket:    # and: 两者都成立
    print("可以入场")

if age < 18 or age > 60:        # or: 至少一个成立
    print("半价")

if not has_ticket:              # not: 取反
    print("请买票")
```
