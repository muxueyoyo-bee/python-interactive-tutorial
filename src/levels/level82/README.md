## 维度灾难

特征越多, 数据越稀疏。100个样本在2维空间里很密, 在100维空间里几乎就是孤立的点 -- 任何模型都难以学习。PCA 解决这个问题。

## PCA 原理

PCA (主成分分析) 找到数据方差最大的方向(主成分), 把数据投影到这些方向上, 丢弃方差小的方向。

```python
from sklearn.decomposition import PCA

pca = PCA(n_components=2)     # 从4维降到2维
X_2d = pca.fit_transform(X_4d)

# explained_variance_ratio_: 每个主成分保留了原始数据多少方差
print(pca.explained_variance_ratio_)  # [0.72, 0.23] -> 保留了95%信息
```

## 核心步骤

1. 数据中心化(减均值)
2. 计算协方差矩阵
3. 特征值分解, 取最大的k个特征向量
4. 原始数据投影到k个特征向量上

## 什么时候用PCA?

- 数据维度太高, 训练太慢
- 想去噪(小方差方向通常是噪声)
- 可视化(降到2D/3D绘图)
- 去除特征之间的相关性
