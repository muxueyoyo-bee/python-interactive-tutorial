# 第五十一关: 过拟合与正则化

过拟合: 模型在训练集上表现完美但无法泛化到新数据.

## 正则化解决方案

- **L1 正则化**: sum(|w|), 产生稀疏权重
- **L2 正则化**: sum(w^2), 让权重趋向较小的值

```python
import numpy as np
w = np.array([0.5, -2.1, 3.7, 0.3, -1.5])
l1 = np.sum(np.abs(w))
l2 = np.sum(w ** 2)
```
