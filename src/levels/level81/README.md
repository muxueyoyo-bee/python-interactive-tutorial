# 第八十一关: SVM -- 最大间隔分类器

> 前置回顾: 第48关 逻辑回归(分类), 第76关 感知机(线性分类)

## SVM 的核心思想

感知机和逻辑回归找一条"能分开"的线。SVM 找一条"分得最开"的线 -- 最大化两类之间的间隔(margin)。

```
     o    |    x
   o      |      x
 o    o   |   x    x
----------+----------  <- 最优决策边界(最大间隔)
          |
```

支持向量(Support Vectors)就是离决策边界最近的那些点, 它们"撑起"了整个分类面。

## 核技巧 (Kernel Trick)

当数据线性不可分时, 用核函数把数据"投射"到高维空间, 在那里找线性边界:

- **线性核**: 原空间直接分类 (数据线性可分时用)
- **RBF核**: 高斯核, 能处理大部分非线性问题 (最常用)
- **多项式核**: 特定模式的非线性

```python
SVC(kernel="linear")  # 线性核
SVC(kernel="rbf")     # RBF核 (默认)
```

> RBF = Radial Basis Function, 本质上是在计算数据点之间的"相似度"。

## SVM vs 逻辑回归

| | SVM | 逻辑回归 |
|------|-----|---------|
| 目标 | 最大化间隔 | 最小化交叉熵 |
| 输出 | 类别 | 概率 |
| 核方法 | 支持 | 不支持 |
| 大数据 | 慢 | 快 |
