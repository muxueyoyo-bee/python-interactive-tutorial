## 为什么需要激活函数?

没有激活函数 = 纯线性变换。多层线性变换 = 一层线性变换。加非线性激活才能拟合任意函数。

## 四大激活函数对比

| 函数 | 公式 | 值域 | 优点 | 缺点 |
|------|------|------|------|------|
| **Sigmoid** | 1/(1+e^(-x)) | (0,1) | 平滑可导, 输出可当概率 | 梯度消失, 非零中心 |
| **Tanh** | (e^x-e^(-x))/(e^x+e^(-x)) | (-1,1) | 零中心, 比sigmoid好 | 仍有梯度消失 |
| **ReLU** | max(0,x) | [0,inf) | 计算快, 缓解梯度消失 | x<0时梯度为0(死神经元) |
| **LeakyReLU** | max(0.01x, x) | (-inf,inf) | 解决死神经元问题 | 多一个超参数 |

## 使用建议

- 隐藏层默认用 **ReLU** (快速, 效果好)
- 二分类输出层用 **Sigmoid**
- 多分类输出层用 **Softmax** (下一关会讲)
- 回归输出层不用激活函数

## ReLU 为什么流行?

```python
# sigmoid: 需要算 exp, 慢
# ReLU: 只是一个 if 判断, 快!

# 梯度:
# sigmoid: 最大0.25, 多层后趋近0(梯度消失)
# ReLU: 要么0, 要么1, 梯度传播稳定
```
