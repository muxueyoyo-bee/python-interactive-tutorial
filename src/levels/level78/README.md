## 反向传播是什么?

反向传播(Backpropagation)是训练神经网络的核心算法 -- 计算损失函数对每个参数的梯度, 然后沿梯度反方向更新参数。

## 三步走

1. **前向传播**: 输入 -> 网络 -> 预测 -> 损失
2. **反向传播**: 损失 -> 逐层回传梯度(链式法则)
3. **参数更新**: w_new = w_old - learning_rate * gradient

## 链式法则手算

```
损失 L = (y_pred - y_true)^2
y_pred = sigmoid(w * x)

dL/dw = dL/dy_pred * dy_pred/dz * dz/dw
      = 2(y_pred-y_true) * y_pred(1-y_pred) * x
```

> 回顾第49关: 梯度下降告诉你"往哪个方向走"。反向传播告诉你"每个参数各自的梯度是多少"。

## 计算图视角 (参考 Karpathy micrograd)

每个运算节点都知道:
- 前向: 给定输入, 计算输出
- 反向: 给定输出的梯度, 计算输入的梯度

这正是 PyTorch/TensorFlow 的 autograd 原理!

## 常见问题

- **梯度消失**: sigmoid两端梯度接近0, 深层网络无法训练 -> 用ReLU
- **梯度爆炸**: 梯度累积过大 -> 用梯度裁剪
- 反向传播需要前向计算的所有中间结果, 内存开销大
