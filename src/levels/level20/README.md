# 第二十关: 文件读取

使用 open() 函数和 with 语句安全地读取文件.

## with 语句 -- 推荐写法

```python
with open("文件路径", "r", encoding="utf-8") as f:
    content = f.read()
    # 文件在这里自动关闭
```

with 的优点: 无论代码正常执行还是出错, 文件都会被自动关闭.

## 读取模式

| 模式 | 含义 |
|------|------|
| "r" | 读取(默认) |
| "w" | 写入(覆盖) |
| "a" | 追加 |
