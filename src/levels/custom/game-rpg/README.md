# 游戏实战: RPG角色战力排行

> 涉及主线知识: 第5关 列表, 第12关 for循环, 第18关 lambda, 第11关 if条件

## 场景

你正在设计一款RPG游戏的角色平衡系统。有5个职业: 战士/法师/射手/坦克/刺客, 每个职业有不同的攻击力、防御力、生命值。

## 战力公式

战力 = 攻击力 + 防御力 + 生命值/100

为什么要除以100? 因为生命值是千位级别, 如果直接相加会完全主导战力, 攻击和防御变得毫无意义 -- 这是**特征缩放**的思想!

## sorted 排序

```python
# 按战力降序排列
ranked = sorted(heroes, key=lambda h: h['atk']+h['def']+h['hp']//100, reverse=True)
```

- `key=`: 指定排序依据
- `reverse=True`: 降序
- `lambda h:`: 为每个英雄计算战力

> 回顾第18关: lambda是匿名函数, 这里对每个英雄提取战力值用于排序。

## 你的任务

5个英雄按战力降序排名, 打印格式: `名次. 名称 (战力:xxx)`。
