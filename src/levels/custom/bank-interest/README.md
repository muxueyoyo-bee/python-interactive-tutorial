# 金融实战: 银行复利计算器

> 涉及主线知识: 第13关 while循环, 第14关 函数, 第3关 数据类型

## 场景

你存了10000元到银行, 年利率3%, 想知道5年后能拿多少钱。复利的意思是: 每年的利息加入本金, 下一年利息基于新本金计算。

## 复利公式

第N年末 = 本金 * (1+利率)^N

```python
year1: 10000 * 1.03 = 10300
year2: 10300 * 1.03 = 10609
year3: 10609 * 1.03 = 10927.27
```

## 单利 vs 复利

- 单利: 每年利息不变 (10000*0.03=300, 5年=11500)
- 复利: 利息滚利息 (5年=11592.74, 多了92.74)

时间越长差距越大 -- 这就是"复利是世界第八大奇迹"的原因。

> 回顾第13关 while: 逐年循环计算, n从1到years, 每年乘以(1+rate)。
