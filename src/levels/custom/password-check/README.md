# 安全实战: 密码强度检测器

> 涉及主线知识: 第24关 正则表达式, 第11关 if条件, 第14关 函数

## 场景

你在开发一个用户注册系统, 需要实现密码强度检测。密码越复杂越安全——衡量标准是密码包含的字符种类。

## 评分规则

每满足一项+1分:
1. 长度 >= 8
2. 包含小写字母 a-z
3. 包含大写字母 A-Z
4. 包含数字
5. 包含特殊符号 !@#$%^&*

得分映射: 1=弱, 2=一般, 3=中等, 4=强, 5=非常强

## 正则检测

```python
import re
re.search(r"[a-z]", pw)     # 有小写?
re.search(r"[A-Z]", pw)     # 有大写?
re.search(r"\d", pw)        # 有数字?
re.search(r"[!@#$%^&*]", pw) # 有特殊符号?
```

> 回顾第24关: `\d` 匹配数字, `[a-z]` 匹配小写字母。正则是字符串模式匹配的神器。

## 你的任务

实现 check_password() 函数, 返回强度等级。测试: "abc123"(弱)、"Python@2025"(非常强)、"123456"(弱)。
