# 数据实战: 天气统计分析

> 涉及主线知识: 第36关 numpy创建数组, 第37关 numpy运算

## 场景

气象局给了你某城市30天的每日最高温度数据, 需要快速出一份统计报告。用numpy可以几行代码搞定。

## 统计指标

- **最高温**: 30天中最热的一天
- **最低温**: 30天中最冷的一天
- **平均温**: 整体温度水平
- **温差**: 最热和最冷的差距, 反映气候波动

## numpy 统计函数

```python
import numpy as np
temps = np.array([32, 28, 35, ...])

np.max(temps)    # 最大值
np.min(temps)    # 最小值
np.mean(temps)   # 平均值
np.ptp(temps)    # 峰峰值(max-min), 也可直接 max-min
```

> 回顾第37关: numpy的向量化运算一次处理整个数组, 比Python循环快。

## 你的任务

用numpy生成30天的随机温度(15-38度), 计算并打印最高温、最低温、平均温、温差。
