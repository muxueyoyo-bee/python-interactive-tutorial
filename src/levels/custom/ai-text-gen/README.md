# AI实战: 迷你文本生成器

> 涉及主线知识: 第25关 模块与导入(random), 第8关 字典, 第5关 列表

## 场景

用最简单的马尔可夫链实现一个文本生成器。给定一组词之间的转移关系, 从起始词开始随机跳转, 生成一句话。

## 马尔可夫链

核心思想: **下一个词只依赖于当前词**。不需要理解句子含义, 只需要知道"这个词后面通常跟哪个词"。

```python
transitions = {
    "我": ["爱", "喜欢", "想"],
    "爱": ["Python", "编程", "学习"],
    "喜欢": ["写代码", "看书", "运动"],
}
```

## random.choice

```python
import random
next_word = random.choice(transitions[current_word])
```

从候选列表中随机选一个。seed(42)固定随机种子让结果可复现。

> 这是GPT的根本原理简化版! GPT用神经网络学习转移概率, 但本质上也是在做"给定上文, 预测下一个词"。

## 你的任务

定义词转移字典, 实现 generate(start, length) 函数, 生成长度5的两句话并打印。
