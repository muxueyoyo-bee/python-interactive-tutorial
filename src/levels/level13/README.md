## # 第十三关: while 循环
while 循环在条件为真时反复执行代码块.

## 基本语法

```python
while 条件:
    执行代码    # 每轮循环后重新检查条件
```

## 与 for 的区别

- for: 知道循环次数(遍历已知序列)
- while: 不知道次数(按条件判断是否继续)

## 示例: 累加到 10

```python
n = 1
total = 0
while n <= 10:
    total += n
    n += 1        # 关键: 不能忘, 否则死循环!
print(total)      # 55
```

> 一定要确保循环条件最终会变为 False, 否则程序会永远运行.
