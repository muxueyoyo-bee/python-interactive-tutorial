# 第七十七关: 多层神经网络 MLP

> 前置回顾: 第76关 感知机(单层), 第79关 激活函数

## 为什么需要多层?

单层感知机只能画一条直线分类。多层网络可以组合多条直线, 逼近任意复杂边界。

```
输入层(2) -> 隐藏层(4, ReLU) -> 输出层(1)
```

## 关键创新

1. **隐藏层**: 输入和输出之间的中间层, 学习更抽象的特征
2. **非线性激活**: 没有激活函数, 多层等于一层! (线性变换的组合仍是线性)
3. **ReLU**: 最常用的激活函数, f(x)=max(0,x), 简单高效

```python
h = relu(X @ W1 + b1)    # 隐藏层: 线性变换 + 非线性激活
out = h @ W2 + b2         # 输出层: 线性变换
```

## MLP 矩阵运算

```python
# X.shape = (3, 2)  3个样本, 2个特征
# W1.shape = (2, 4)  2->4维变换
# h.shape = (3, 4)  隐藏层输出
# W2.shape = (4, 1)  4->1维变换
# out.shape = (3, 1)  最终预测
```

> 回顾矩阵乘法: `@` 运算符, 在第56关 Self-Attention 里也大量使用。

## 实际中的深度

"深度"学习就来自多层堆叠: 输入->隐藏1->隐藏2->...->隐藏N->输出。每层学不同抽象级别。
