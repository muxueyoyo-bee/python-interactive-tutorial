# 第四十六关: 什么是人工智能

## AI 的三个层次

- **人工智能(AI)**: 让机器模拟人类智能
- **机器学习(ML)**: AI的子集, 从数据中学习规律而非手动编写规则
- **深度学习(DL)**: ML的分支, 使用多层神经网络

三者的关系: AI > ML > DL

## 机器学习的三种类型

| 类型 | 说明 | 例子 |
|------|------|------|
| 监督学习 | 有标注数据 | 分类, 回归 |
| 无监督学习 | 无标注数据 | 聚类, 降维 |
| 强化学习 | 与环境交互学习 | 游戏AI, 机器人 |
