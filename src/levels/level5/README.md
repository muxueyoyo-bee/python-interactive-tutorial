## # 第五关: 列表 - 有序的集合
列表(list)是 Python 最灵活的容器类型, 用于存储一组有序的, 可变的元素.

## 创建列表

```python
fruits = ["苹果", "香蕉", "橘子", "葡萄"]
nums = [1, 2, 3, 4, 5]
mixed = [1, "hello", 3.14, True]  # 可混合不同类型
```

## 索引访问

列表的元素通过索引访问, 索引从 0 开始:

```python
fruits = ["苹果", "香蕉", "橘子", "葡萄"]

print(fruits[0])    # 苹果 (第一个)
print(fruits[1])    # 香蕉 (第二个)
print(fruits[-1])   # 葡萄 (最后一个, 负数从末尾倒着数)
print(fruits[-2])   # 橘子 (倒数第二个)
```
