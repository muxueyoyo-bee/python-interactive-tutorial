## 感知机是什么?

感知机(Perceptron)是神经网络的最基本单元, 1958年由 Rosenblatt 提出。它模拟一个神经元: 接收多个输入信号, 加权求和, 超过阈值就激活输出1, 否则输出0。

```
输入 x1,x2 -> [加权求和 w1*x1+w2*x2+b] -> [阶跃函数] -> 输出 0/1
```

## 数学定义

output = 1 if (w1*x1 + w2*x2 + b > 0) else 0

其中:
- w1, w2: 权重, 控制每个输入的重要程度
- b: 偏置, 控制激活的难易程度(把决策边界向左/右平移)
- 本质上就是一条直线(w1*x1+w2*x2+b=0)把空间分成两半

> 回顾第47关: 线性回归也是 w*x+b, 但回归输出连续值, 感知机输出0/1。

## 感知机实现逻辑门

单个感知机可以实现 AND, OR, NAND 等逻辑门:

```python
def perceptron(x, w, b):
    return 1 if (w[0]*x[0] + w[1]*x[1] + b) > 0 else 0

# AND门: 两个都1才输出1
# 权重[1,1], 偏置-1.5 -> 只有1*1+1*1-1.5=0.5>0 才激活
```

## 感知机的局限性

单层感知机无法解决 XOR(异或)问题 -- 1969年 Minsky 证明了这一点, 导致第一次"AI寒冬"。解决方案就是第77关的多层网络。
