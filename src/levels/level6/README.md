## # 第六关: 列表的增删改查
列表是可变的, 意味着你可以随意添加, 删除和修改其中的元素.

## 增加元素
```python
nums = [1, 2, 3]
nums.append(4)        # 末尾追加
nums.insert(0, 0)     # 在索引0处插入0
```

## 删除元素
```python
nums = [1, 2, 3, 2]
nums.remove(2)        # 删除第一个值为2的元素
popped = nums.pop()   # 弹出最后一个元素
```

## 修改元素
```python
nums = [1, 2, 3]
nums[0] = 99          # 直接赋值修改
```

## 你的任务

从 [1, 2, 3] 开始: append(4) -> insert(0, 0) -> remove(2) -> 打印最终列表.

预期输出:
```
[0, 1, 3, 4]
```
