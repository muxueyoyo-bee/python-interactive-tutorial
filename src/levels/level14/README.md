## # 第十四关: 自定义函数
函数(def)是代码复用的基本单位 -- 把一段逻辑封装起来, 之后可以反复调用.

## 定义函数

```python
def 函数名(参数):
    函数体
    return 返回值
```

## 示例

```python
def greet(name):
    return f"你好, {name}!"

# 调用函数
print(greet("小明"))    # 你好, 小明!
print(greet("小红"))    # 你好, 小红!
```

## 你的任务

定义一个函数 greet(name), 接收名字参数, 返回 "你好, {name}!" 格式的字符串. 分别用 "小明" 和 "小红" 调用并打印.

预期输出:
```
你好, 小明!
你好, 小红!
```
