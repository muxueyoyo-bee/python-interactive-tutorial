# 第八十关: 偏差与方差的权衡

> 前置回顾: 第50关 损失函数(MSE), 第51关 过拟合与正则化, 第47关 线性回归

## 预测误差的三个来源

总误差 = 偏差^2 + 方差 + 不可约误差

- **偏差(Bias)**: 模型平均预测偏离真实值的程度。高偏差=欠拟合
- **方差(Variance)**: 模型对训练数据波动的敏感程度。高方差=过拟合
- **不可约误差**: 数据本身的噪声, 任何模型都无法消除

## 三种模型的直观对比

```python
# 真实关系: y = 2x + 1 + 噪声

# 模型1: 常数 (高偏差, 低方差)
# 预测永远是一个常数, 过于简单

# 模型2: 10次多项式 (低偏差, 高方差)
# 完美拟合训练点, 但在新数据上波动剧烈

# 模型3: 一次多项式 (偏差和方差平衡)
# 接近真实关系, 泛化能力好
```

> 回顾第51关: L1/L2正则化就是在偏差和方差之间找平衡 -- 加惩罚增加偏差但降低方差。

## 偏差-方差权衡图

```
模型复杂度 ->
  低复杂度: 高偏差, 低方差 (欠拟合)
  中等复杂度: 最优平衡点
  高复杂度: 低偏差, 高方差 (过拟合)
```

## 如何诊断?

- 训练误差高 + 验证误差高 = **欠拟合**(高偏差) -> 增加模型复杂度
- 训练误差低 + 验证误差高 = **过拟合**(高方差) -> 加正则化/更多数据
- 训练误差低 + 验证误差低 = **刚好**
