# 第四十七关: 线性回归 - 从零实现

线性回归是最基础的监督学习算法, 用一条直线拟合数据点.

## 核心公式

y = wx + b, 其中 w 是权重(斜率), b 是偏置(截距).

## 最小二乘法

对于过原点的直线 y = wx, 最优解为: w = sum(x*y) / sum(x*x)

```python
import numpy as np
X = np.array([1, 2, 3, 4, 5])
y = np.array([2, 4, 6, 8, 10])
w = np.sum(X * y) / np.sum(X * X)
print(f"拟合斜率 w = {w:.2f}")
```
