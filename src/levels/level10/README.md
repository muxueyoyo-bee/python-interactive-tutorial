## # 第十关: 集合 - 不重复的元素
集合(set)是一个无序, 不重复的元素集合, 和数学中的集合概念完全对应.

## 创建集合

```python
s1 = {1, 2, 3, 4}
s2 = {3, 4, 5, 6}

# 用 set() 去重
nums = [1, 1, 2, 2, 3]
unique = set(nums)    # {1, 2, 3}
```

## 集合运算

Python 集合原生支持数学集合运算:

| 运算 | 符号 | 含义 |
|------|------|------|
| 交集 | & | 两个集合都有的元素 |
| 并集 | \| | 两个集合的全部元素 |
| 差集 | - | 在第一个但不在第二个的元素 |

```python
s1 = {1, 2, 3, 4}
s2 = {3, 4, 5, 6}

print(s1 & s2)    # {3, 4}         交集
print(s1 | s2)    # {1,2,3,4,5,6}  并集
print(s1 - s2)    # {1, 2}         差集
```
