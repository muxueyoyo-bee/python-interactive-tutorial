# 第十关: 集合 -- 不重复的元素

> 前置回顾: 第5关 列表可重复, 第8关 字典键不可重复

## 集合是什么?

集合(set)结合了:
- 像列表: 存多个元素
- 像字典键: 元素不可重复
- 像数学集合: 支持交/并/差集

```python
s1 = {1, 2, 3, 4}
s2 = set([1, 1, 2, 2, 3])   # {1, 2, 3} -- 自动去重!

# 空集合必须用 set(), {} 是空字典!
empty = set()
```

## 核心用途: 去重

```python
numbers = [1, 1, 2, 2, 3, 3, 4]
unique = list(set(numbers))     # [1, 2, 3, 4]
```

> 回顾第5关: 列表允许重复。当你需要"不重复"时, 用集合。

## 集合运算

```python
s1 = {1, 2, 3, 4}
s2 = {3, 4, 5, 6}

print(s1 & s2)    # {3, 4}       交集: 都有的
print(s1 | s2)    # {1,2,3,4,5,6} 并集: 全部的
print(s1 - s2)    # {1, 2}       差集: 只在s1的
```

## 实际场景

```python
customers_A = {"张三", "李四", "王五"}
customers_B = {"王五", "赵六", "张三"}

both = customers_A & customers_B    # 两边都买的客户
only_A = customers_A - customers_B  # 只买A的客户
```

## 集合的限制

- 元素必须不可变 (不能存列表和字典)
- 无序 (不能用索引, 不能 s[0])
