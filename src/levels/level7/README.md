# 第七关: 元组 -- 不可变的序列

> 前置回顾: 第5关 列表创建和索引, 第6关 列表是可变的

## 元组 vs 列表

| 特性 | 列表 (list) | 元组 (tuple) |
|------|------------|-------------|
| 符号 | `[ ]` | `( )` |
| 可变性 | 可增删改 | 不可修改 |
| 速度 | 稍慢 | 稍快 |
| 做字典键 | 不能 | 能 |

## 创建元组

```python
t = (1, 2, 3, "Python")    # 标准写法
empty = ()                   # 空元组
single = (1,)                # 单元素! 逗号不能省!
not_tuple = (1)              # 这是整数 1, 不是元组!
```

## 为什么用元组?

1. 保护数据不被意外修改 (数据库配置、常量)
2. 函数返回多个值 `return x, y` 本质返回元组
3. 作为字典的键 (列表不能, 元组能)

## 访问元素

和列表完全一样, 只是不能改:
```python
t = (1, 2, 3, "Python")
print(t[0])      # 1
print(t[-1])     # "Python"
print(len(t))    # 4

# t[0] = 99      # 报错! 元组不允许修改
```

> 回顾: 索引规则和第5关列表完全一样, 从0开始, 负数倒着数。

## 常见错误

- 忘了逗号: `(1)` 是整数 1, `(1,)` 才是元组
- 试图修改元组: TypeError, 用 `list(t)` 转成列表再改
