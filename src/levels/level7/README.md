## # 第七关: 元组 - 不可变的序列
元组(tuple)和列表很像, 但有一个关键区别: 元组是不可变的, 一旦创建就不能修改.

## 创建元组

```python
t = (1, 2, 3, "Python")       # 用圆括号
empty = ()                      # 空元组
single = (1,)                   # 单元素元组, 逗号不能省!
```

## 为什么需要元组?

- 安全: 数据不会被意外修改
- 性能: 比列表稍快, 内存占用更少
- 字典键: 元组可以作为字典的 key (列表不能)

## 访问元素

和列表一样用索引:
```python
t = (1, 2, 3, "Python")
print(t[0])     # 1
print(t[-1])    # "Python"
print(len(t))   # 4
```
