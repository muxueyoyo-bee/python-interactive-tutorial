# 第十七关: 字典推导式

和列表推导式类似, 字典推导式用一行代码创建新字典.

## 基本语法

```python
{键表达式: 值表达式 for 变量 in 可迭代对象}
```

## 示例

```python
squares = {x: x**2 for x in range(1, 6)}
print(squares)
# {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}
```

## 你的任务

用字典推导式生成 {1: 1**2, 2: 2**2, ..., 5: 5**2} 并打印.

预期输出:
```
{1: 1, 2: 4, 3: 9, 4: 16, 5: 25}
```
